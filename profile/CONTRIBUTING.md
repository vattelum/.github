# Contributing to Vattelum

Vattelum is open-source infrastructure for decentralized lawmaking. 
The software is intentionally simple. Its value comes from real-world deployment.

## The Best Way to Contribute

**Fork it. Deploy it. Create law with it.**

Start with creating your own registry, and start building your jurisdiction.
That is what this project is for.

## Bug Reports & Suggestions

Found a bug or have a suggestion or suggestion? Open an Issue in the relevant repository.
All issues are reviewed by the maintainer.

## Pull Requests

Pull requests are welcome but should be kept minimal and focused. 
The goal is to keep the infrastructure lean and interoperable.
All PRs require maintainer approval before merging.

## What Lives Where

- **This repository** — source code only
- **On-chain** — the laws, standards, and contracts created using this software

Do not submit legal standards or documents via GitHub. 
They belong on-chain, in a deployed registry instance (plain, BVS or DAA).
